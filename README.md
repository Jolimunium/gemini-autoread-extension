# gemini-autoread-extension

## ✨ Overview

**Gemini Auto-Read** is a Chrome Extension that automatically plays the voice response on [Gemini](https://gemini.google.com) whenever a new answer appears.

## 🚀 Features

- **Auto-play** the audio response as soon as Gemini finishes generating an answer.
- **Toggle** the feature on/off via a floating UI button.
- **Sync** the setting across tabs and browsers.
- **Keyboard shortcuts** for quick enable/disable.
- **Debug log export** for troubleshooting.

## 📦 Installation

Install the extension directly from the **Chrome Web Store**:

[👉 **Get it on Chrome Web Store**](https://chrome.google.com/webstore/detail/your-extension-id)

## 🛠️ Usage

- Click the **Floating Toggle** button (bottom-right) to enable/disable auto-read.
- Use the shortcut **Ctrl+Shift+E** (default) to toggle the feature.
- Open the **Debug panel** (gear icon) to view logs and export them as a JSON file.
